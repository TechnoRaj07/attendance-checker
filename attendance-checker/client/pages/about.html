PROJECT OVERVIEW

The University Student Attendance Checker is a production-ready, full-stack web application designed to solve the universal student dilemma: maintaining the mandatory 75% attendance criteria.

Originally engineered in C++, this system has been completely rewritten into a modern JavaScript stack while preserving the exact logical precision of the original mathematical formula.

CORE OBJECTIVES

  • Provide 100% accurate attendance predictions.
  • Generate official PDF records for administrative submission.
  • Deliver a premium, futuristic user experience (Cyberpunk UI).
  • Supply administrators with deep analytical insights.

THE MATHEMATICS

// Original C++ Implementation
float percentage = (attended / total) * 100;
int x = (3 * total) - (4 * attended);
if (x < 0) x = 0;
Why 3*Total - 4*Attended?
This derived formula calculates the exact number of consecutive classes a student must attend to reach 75%.

(Attended + x) / (Total + x) = 0.75
Attended + x = 0.75 * Total + 0.75 * x
0.25 * x = 0.75 * Total - Attended
x = 3 * Total - 4 * Attended
